# AI Videos

Auto-generated list of every literature note tagged `#type/source/video` (or any nested video subtype like `#type/source/video/ai-llms`).

This is a Dataview query — requires the [Dataview plugin](https://github.com/blacksmithgu/obsidian-dataview) (pre-registered in `.obsidian/community-plugins.json`).

```dataview
LIST
FROM #type/source/video
SORT file.name ASC
```

---

## How this works

You don't hand-maintain this list. As soon as you ingest a video and the resulting literature note carries `#type/source/video` in its tags, it appears here.

Rename this file (and the corresponding `index.md` entry) if your video corpus isn't AI-focused — e.g. `System Design Videos.md` filtered on `#type/source/video/system-design`.
